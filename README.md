# L’étiquettix 3000

Générateur local d’étiquettes camping au format Lyreco 70 × 37 mm, 24 étiquettes par page A4.

## Lancer en développement

```bash
npm install
npm run dev
```

## Générer la version web

```bash
npm run build
```

La V10 refait le design de l’accueil et des étiquettes, ouvre le PDF en aperçu, garde les dimensions millimétrées et ajoute l’easter egg sur le logo/titre.
